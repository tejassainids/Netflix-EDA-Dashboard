# 🎬 Netflix Content Analysis Dashboard

A complete Exploratory Data Analysis (EDA) project on the Netflix Titles dataset using **Python, Pandas, Matplotlib, and Seaborn**.

---

## 📌 Project Overview

This project explores Netflix's content catalog to uncover trends in movies, TV shows, genres, countries, release years, and audience ratings. The analysis concludes with a dashboard that summarizes the key findings.

---

## 🛠️ Tech Stack

- Python
- Pandas

## 💡 Business Insights

### 🎬 Content Type
- Movies dominate Netflix's catalog.
- TV Shows form a smaller but significant portion of the library.

### 🌍 Global Reach
- The United States contributes the highest number of titles.
- India and the United Kingdom are among the leading content producers.

### 🎭 Genre Distribution
- International Movies and Dramas are the most common genres.
- Netflix emphasizes globally appealing content.

### 📅 Content Addition
- Content additions vary throughout the year, indicating seasonal release strategies.

### 📺 Audience Ratings
- Mature audience content (TV-MA) has increased considerably over time.

### 📈 Release Trend
- Most available titles were released in recent years, reflecting Netflix's focus on newer content.
